# exam-seating-plan
A Python-based automation tool for generating randomized school exam seating plans using MySQL database integration and custom shuffling logic.

Project Overview
This School Exam Seating Arrangement System is designed to simplify the tedious task of manually assigning seats for school examinations. By fetching student data directly from a MySQL database, the system randomizes the student list and intelligently distributes them across available rooms and rows.

Key Features:
Database Integration: Securely connects to MySQL to fetch real-time student records.
Randomization: Uses Python's random module to ensure fair and unbiased seating.
Customizable Layouts: Allows users to define the number of rooms, desks per room, and rows per room.
Automated Reporting: Generates a formatted seating_plan.txt file ready for printing.
